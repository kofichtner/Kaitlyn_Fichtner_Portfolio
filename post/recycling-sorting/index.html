Recycling Sorting Machine Learning Algorithm

January 1, 2021

Recycling and composting rates have been steadily increasing from 6.0% of total Municipal Solid Waste (MSW) in 1960 to 32.1% of MSW in 2018 [1]. With a growing amount of waste being recycled, automating the process of sorting recycling would make recycling facilities more efficient and improve conditions for workers. Recycling workers are at risk for severe injuries including loss of limb if caught between moving machinery, being crushed by equipment, injuries from falls, and hand or finger amputation or laceration from sharp objects [2,3]. Recycling workers also face long term risks from exposure to hazardous dust, chemicals, materials, and cancer-causing compounds [2,3]. The goal of this project is to develop a recycling sorting method that can be used by an industrial recycling sorting robot that has been developed in an ongoing project at Worcester Polytechnic Institute.

This project is in early stages and still in progress. The following results are what I’ve learned so far.

Summary:

  • Convolutional Neural Network for image classification
  • Support Vector Machine for image classification

Results:

In January 2021, I began testing machine learning algorithms using the OpenRecycle dataset to classify recycling images into 7 classes. Initial results from a SVM model and a CNN model are shown below. The SVM model achieved an accuracy of ~50% and the CNN had an accuracy of ~20%, as seen in the confusion matricies below. These results indicate a change in approach is needed. A next step will be experimenting with an R-CNN model using the TACO dataset. However, the concern with an R-CNN is that it would not be fast enough to detect many objects at once on a moving conveyer belt of recycling, so using a fast-R-CNN or a faster-R-CNN will be necessary. Improving the feature extraction methods may also be a viable way to enhance the performance of the SVM.
